Rubrik RSC RBS Agent Version ReporterA PowerShell utility designed to query Rubrik Security Cloud (RSC) via GraphQL to report on Rubrik Backup Service (RBS) agent versions across Windows and Linux/Unix physical hosts.FeaturesUnified Reporting: List all physical hosts and their active RBS versions in a single view.Smart Outlier Detection: Automatically flags hosts with "8.x" versions or "Unknown" statuses for manual validation.Interactive HTML Export: Generates a sortable, professional HTML report.CSV Export: Includes execution metadata (timestamp, instance URL, and service account name).PrerequisitesPowerShell 7.x: This script is optimized for PowerShell 7. Core PowerShell 5.1 is not recommended.RubrikSecurityCloud Module:Install-Module -Name RubrikSecurityCloud -Scope CurrentUser
RSC Service Account:Create a Service Account in your RSC tenant.Role: Assign a role with at least Global Read-Only permissions.JSON Key: Download the Service Account JSON key file.Setup & Security1. Encrypt Your Service Account KeyTo avoid leaving plain-text credentials on your machine, it is highly recommended to encrypt your Service Account JSON file using the SDK's built-in utility:Protect-RscServiceAccountFile -Path "C:\path\to\your-raw-key.json" -OutPath "C:\scripts\json\Shane-SA-Encrypted.json"
The script will use this encrypted file to authenticate.2. Prepare the ScriptEnsure Get-RscHostRbsVersion.ps1 is saved to your local scripts folder.UsageRun the script by providing the path to your encrypted JSON file and an export directory:.\Get-RscHostRbsVersion.ps1 -ServiceAccountFile "C:\scripts\json\Shane-SA-Encrypted.json" -ExportPath "C:\reports"
Manual Validation RecommendedThe script adds a specific note to the "Manual Validation Recommended" column if:A host is running version 8.x.The upgrade status is UPGRADE_STATUS_UNKNOWN.The version returns as Unknown.LicenseThis project is licensed under the MIT License - see the LICENSE file for details.
